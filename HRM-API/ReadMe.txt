1. Added MS.EFCore - Modren Object-database mapper
	------------------------------------------------------------------------------------
	Entity Framework Core is a modern object-database mapper for .NET.
	It supports LINQ queries, change tracking, updates, and schema migrations.
	EF Core works with SQL Server, Azure SQL Database, 
						SQLite, Azure Cosmos DB, MySQL,
						PostgreSQL, and other databases through a provider plugin API.

	Commonly Used Types:
	Microsoft.EntityFrameworkCore.DbContext
	Microsoft.EntityFrameworkCore.DbSet

	------------------------------------------------------------------------------------

2. Added MS.EFCore.Tools 
	------------------------------------------------------------------------------------
	Entity Framework Core Tools for the NuGet Package Manager Console in Visual Studio.

		Enables these commonly used commands:
		Add-Migration
		Bundle-Migration
		Drop-Database
		Get-DbContext
		Get-Migration
		Optimize-DbContext
		Remove-Migration
		Scaffold-DbContext
		Script-Migration
		Update-Database
	--------------------------------------------------------------------------------------

3. Added MS.EFCore.SqlServer 
	--------------------------------------------------------------------------------------
	Microsoft SQL Server database provider for Entity Framework Core.
